# React + Vite

Este proyecto está hecho con JavaScript, React, CSS y HTML.
Contiene funcionalidades como ordenar alfabéticamente por países, filtrar por país (estén ordenados o no) y permitir el filtrado incluso cuando los países no están ordenados.

Además, incluye botones que permiten borrar usuarios, colorear las filas con dos colores diferentes según si se encuentran en índices pares o impares, y resetear el estado inicial después de haber realizado diversas operaciones, si así se desea.

Separé el useEffect y el fetch en un custom hook llamado useUsers.
Se utiliza componetización también.
